동해 다온스테이 사이트 실행 방법
================================

Windows에서 직접 실행:

1. '서버실행.bat' 파일을 더블클릭
2. 명령창이 열리고 서버가 시작됩니다
3. 브라우저에서 http://localhost:8080 접속
4. 완전한 React 사이트가 실행됩니다

또는 수동 실행:

1. 명령프롬프트(cmd) 실행
2. 다음 명령어 입력:
   cd F:\confac2025\confac2025.github.io_v2\dist
   python -m http.server 8080
3. 브라우저에서 http://localhost:8080 접속

포함된 내용:
- Hero Section (영상 배경)
- Video Section  
- ROI Analysis
- Tourist Stats
- TourInfraMap Section (위성지도 + 향상된 마커)
- Why Invest
- Growth Potential
- Form Section
- Applicant List
- Project Overview
- Gallery Section
- Footer

모든 기능이 포함된 완전한 투자 안내 사이트입니다.
